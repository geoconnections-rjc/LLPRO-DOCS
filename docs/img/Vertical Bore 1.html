You have selected to use a custom borehole resistance. You must provide an independent borehole resistance for both heating and cooling. These resistance values will be used under the assumption that they include:

  1. Fluid film resistance
  2. Pipe wall resistance
  3. Grout resistance through the bore annulus.
  4. Resistance of permanent casing.

It is also assumed that the resistance value has been properly adjusted to account for the location of the custom pipe configuration with respect to the bore wall.

You will not be able to use the headering function for custom borehole resistances.
